# RELATORIOS

Hospital reporting dashboard built with HTML, JavaScript, Google Apps Script, and Google Workspace.

## Overview

This project was developed to solve real operational problems in healthcare environments using web technologies and Google Workspace tools. It provides a comprehensive solution for generating analytical CRP reports with data organized in Google Sheets.

## Features

- Dashboard interface
- - Analytical reporting
  - - Data aggregation and filtering
    - - Real-time KPI monitoring
      - - Responsive design
        - - Google Workspace integration
          - - Multi-user access
           
            - ## Technologies
           
            - - JavaScript
              - - HTML
                - - CSS
                  - - Google Apps Script
                    - - Google Sheets
                      - - Google Web Apps
                       
                        - ## Purpose
                       
                        - The goal of this project is to improve operational efficiency in healthcare settings by providing automated reporting solutions, reducing manual work, and supporting better decision-making through clear data visualization.
